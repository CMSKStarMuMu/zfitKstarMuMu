# zfitKstarMuMu
 KstarMuMu fit code using zfit (both Si and Pi base)


 Python version:3.8.5

 packages using in the code:
 zfit 0.5.4
 tensorflow 2.2.0
 root-numpy 4.8.0
 matplotlib 3.3.2

 HOW TO USE
 
 GEN-LEVEL fit
 python genfit.py q2Bin pdfbase minimizer
 q2Bin:0,1,2,5,7
 pdfbase:Si Pi
 minimizer:minuit,scipy,adam(now only minuit is available)
